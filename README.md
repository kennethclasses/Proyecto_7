# Proyecto_7
En este primer proyecto simulamos trabajar para una tienda online que vende videojuegos con alcance global. Tomamos en cuenta las reseñas de usuarios y expertos, los géneros, las plataformas y los datos históricos sobre las ventas de juegos que están disponibles en fuentes abiertas. 

Autor: Kenneth Toral
Escrito en VS code, Python.
